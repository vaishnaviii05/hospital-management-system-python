# 🏥 Hospital Management System (Python + Flask)

## 📌 Project Overview

The **Hospital Management System** is a web-based application developed using **Python and Flask** that helps hospitals manage patient records, doctor information, and hospital operations efficiently.

This system provides APIs and backend logic to handle hospital-related data and perform operations such as managing patient information and medical records.

The project uses **SQLite database** for storing hospital data and **Flask-RESTful** for building REST APIs.

---

## 🚀 Features

* Patient management
* Doctor information management
* REST API using Flask
* SQLite database integration
* JSON configuration support
* Simple hospital management backend system

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/hospital-management-system-python.git
```

### 2️⃣ Navigate to project directory

```
cd hospital-management-system-python
```

### 3️⃣ Install dependencies

```
pip install flask flask-restful flask-cors
```

### 4️⃣ Run the application

```
python app.py
```

---

## 🌐 Running the Application

After running the server, open your browser and go to:

```
http://127.0.0.1:5000
```
